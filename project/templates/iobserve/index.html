eclipt.is, a service by onekilopars.ec

The goal is to provide modern web access to astronomical information, that is, browsable APIs where each resource has its own (that is, unique) Uniform Resource Locator (a.k.a. URL). Astronomy has often been in the forefront of organising its big data on the web, making available tons of useful information. However, contact with the web culture is not strong enough, and makes this information rarely accessible otherwise than through web pages, i.e. readable mostly by humans.

The ambition behind eclipt.is is to bring some of the web culture to that information, and provides RESTful APIs for various sources of astronomical information, thus enabling to build new apps. As a former professional astronomer, I've written tons of (Python) scripts. Scripts are good, but tend to generate more scripts. That's one of the reason I've built iObserve (also for iPad). An app illustrates the value of integration.
Logically, eclipt.is is also built for being the backend of the coming iObserve 2.

The scope of eclipt.is is potentially enormous. This project is meant to evolve in the coming years.

June 2015.
